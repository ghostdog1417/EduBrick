# EduBrick

EduBrick is a static educational website for students preparing for JEE, NEET, board exams, and computer-related courses. It includes a landing page, login/register screen, subject pages, sample papers, quiz content, and supporting assets.

## Features

- Homepage with featured subjects and service highlights
- Login and registration UI
- Subject-specific pages for JEE, quizzes, computer courses, and sample papers
- Responsive styling and image-based content sections

## Project Structure

- `index.html` - main landing page
- `login.html` - login and registration page
- `style.css` - site styles for the landing page
- `loginStyle.css` - styles for the login page
- `script.js` - shared page interactions
- `images/` - icons, illustrations, and creator photos
- `subjects/` - subject and quiz pages
- `samplePapers/` - sample paper content

## How To Run

1. Open `index.html` directly in a browser, or
2. Use a local static server such as VS Code Live Server for the best experience.

## Notes

- This project does not require a build step or package install.
- External scripts on the pages rely on an internet connection, including jQuery from a CDN.
